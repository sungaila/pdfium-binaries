This is a fork of [bblanchon/pdfium-binaries](https://github.com/bblanchon/pdfium-binaries) that provides [PDFium](https://pdfium.googlesource.com/pdfium/) builds for Windows [linking the umbrella library OneCore.lib](https://learn.microsoft.com/en-us/windows/win32/apiindex/windows-umbrella-libraries). These builds work on Windows 10 (or newer) and other product families like Nano Server.
